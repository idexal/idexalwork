# IdexalWork CLI

The official command-line interface for IdexalWork — professional AI agent desktop environment.

## Installation

```bash
# Via npm (recommended for developers)
npm install -g @idexal/idexalwork-cli

# Or download from GitHub Releases
# https://github.com/idexal/idexalwork/releases
```

The CLI is also installed automatically alongside the IdexalWork Desktop installer.

## Usage

```bash
idexalwork --help
idexalwork --version
```

## Building from source

```bash
git clone https://github.com/idexal/idexalwork.git
cd idexalwork
npm install
npm run build
```

## Release

Binaries are built for:

- **Windows** (x64) — `.exe`
- **macOS** (arm64, x64) — `.tar.gz`
- **Linux** (x64, arm64) — `.tar.gz`
